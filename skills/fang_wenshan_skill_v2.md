# 方文山体 Generation Skill

你正在以方文山的词风创作歌词或诗句。
方文山体最核心的特质：用古典意象和器物作为情感的载体，从不直接说出情感本身。

## 风格指纹

【风格指纹】

1. 词汇与用语：
   - 高频出现的词汇类型包括古典意象词汇和情感词汇，如“檀香”、“宣纸”、“仕女图”、“菊花”、“东风”、“狼牙月”、“伽蓝寺”等。
   - 古汉语词汇与现代语法的混搭：使用古典词汇搭配现代语法结构，如“素胚勾勒出青花笔锋浓转淡”中的“素胚”、“青花”、“笔锋”。
   - 例句：“素胚勾勒出青花笔锋浓转淡”，“狼牙月伊人憔悴”，“雨纷纷旧故里草木深”。

2. 句法与结构：
   - 句式长短节奏规律：以短句为主，偶尔穿插长句，形成节奏上的变化。
   - 对仗/排比的具体形式：使用对仗，如“花落人断肠我心事静静躺”。
   - 行内断句：常在句中使用逗号或顿号进行断句，如“你发如雪，纷飞了眼泪”。

3. 意象体系：
   - 核心意象群：自然景物（如“月”、“花”、“雨”）、古典物品（如“青花瓷”、“琵琶”）、历史文化（如“伽蓝寺”、“仕女图”）。
   - 意象之间的转接：通过情感或动作连接，如“狼牙月伊人憔悴，我举杯饮尽了风雪”。
   - 跨行意象链的具体例子：“帘外芭蕉惹骤雨门环惹铜绿，而我路过那江南小镇惹了你”。

4. 叙事视角与情感表达：
   - 情感通过物象间接承载：使用物象代替直接的情感陈述。
   - 例子1：“素胚勾勒出青花笔锋浓转淡”——通过青花瓷的描绘表达细腻情感。
   - 例子2：“菊花残满地伤你的笑容已泛黄”——通过菊花的凋零传达悲伤。
   - 例子3：“雨纷纷旧故里草木深”——通过雨和旧故里的描写传达孤独。

5. 独特句法机制：
   - 机制1：古典意象的现代化组合
     - 引用：“素胚勾勒出青花笔锋浓转淡”，“狼牙月伊人憔悴”。
     - 说明：将古典意象与现代情感结合，创造出独特的诗意氛围。
   - 机制2：情感的物象化表达
     - 引用：“菊花残满地伤你的笑容已泛黄”，“雨纷纷旧故里草木深”。
     - 说明：通过物象传达复杂情感，使情感表达更具层次感。
   - 机制3：时间与空间的交错叙述
     - 引用：“一盏离愁孤单伫立在窗口”，“岁月在墙上剥落看见小时候”。
     - 说明：通过时间与空间的交错，营造出怀旧与思念的氛围。

6. 此风格刻意回避的东西：
   - 缺席的句型：直接的情感陈述和简单的叙述句型。
   - 缺席的词汇类型：现代口语词汇和直接的情感词汇。
   - 缺席的情感表达方式：直接的情感宣泄和简单的叙事结构。

## 正例

### 正例：青花瓷
素胚勾勒出青花笔锋浓转淡
瓶身描绘的牡丹一如你初妆
冉冉檀香透过窗心事我了然
宣纸上走笔至此搁一半

釉色渲染仕女图韵味被私藏
落款时却惦记着你
你隐藏在窑烧里千年的秘密
极细腻犹如绣花针落地

帘外芭蕉惹骤雨门环惹铜绿
而我路过那江南小镇惹了你
在泼墨山水画里你从墨色深处被隐去

**风格特征：** 这首作品中，句子“素胚勾勒出青花笔锋浓转淡”体现了机制1：古典意象的现代化组合。通过将古典的“素胚”和“青花”与现代情感结合，创造出独特的诗意氛围。另一个例子是“帘外芭蕉惹骤雨门环惹铜绿，而我路过那江南小镇惹了你”，通过跨行意象链，营造出怀旧与思念的氛围。

### 正例：菊花台
你的泪光柔弱中带伤
惨白的月弯弯勾住过往
夜太漫长凝结成了霜
是谁在阁楼上冰冷地绝望

菊花残满地伤你的笑容已泛黄
花落人断肠我心事静静躺
北风乱夜未央你的影子剪不断
徒留我孤单在湖面成双

花已向晚飘落了灿烂
凋谢的世道上命运不堪
愁莫渡江秋心拆两半
怕你上不了岸一辈子摇晃

**风格特征：** 句子“菊花残满地伤你的笑容已泛黄”运用了机制2：情感的物象化表达，通过菊花的凋零传达复杂的悲伤情感。另一个例子是“花落人断肠我心事静静躺”，通过花落的意象表达内心的断肠之情，使情感表达更具层次感。

### 正例：东风破
一盏离愁孤单伫立在窗口
我在门后假装你人还没走
旧地如重游月圆更寂寞
夜半清醒的烛火不忍苛责我

一壶漂泊浪迹天涯难入喉
你走之后酒暖回忆思念瘦
水向东流时间怎么偷
花开就一次成熟我却错过

谁在用琵琶弹奏一曲东风破
岁月在墙上剥落看见小时候
犹记得那年我们都还小小的
你用背包装着所有我却背着你

**风格特征：** 句子“一盏离愁孤单伫立在窗口”体现了机制3：时间与空间的交错叙述，通过将离愁与空间结合，营造出怀旧的氛围。另一个例子是“岁月在墙上剥落看见小时候”，通过时间的交错，表达对过去的思念。

### 正例：发如雪
狼牙月伊人憔悴
我举杯饮尽了风雪
是谁打翻前世柜
惹尘埃是非

缘字诀几番轮回
你锁眉哭红颜唤不回
纵然青史已经成灰
我爱不灭

繁华如三千东流水
我只取一瓢爱了解
只恋你化身的蝶

你发如雪纷飞了眼泪
我等待苍老了谁
红尘醉微醺的岁月
我用无悔刻永恒

**风格特征：** 句子“狼牙月伊人憔悴”使用了机制1：古典意象的现代化组合，将“狼牙月”与“伊人”结合，创造出浓厚的诗意氛围。另一个例子是“你发如雪纷飞了眼泪”，通过发如雪的意象表达复杂的情感，使情感表达更具层次感。

### 正例：烟花易冷
雨纷纷旧故里草木深
我听闻你始终一个人
斑驳的城门盘踞着老树根
石板上回荡的是再等

伽蓝寺听雨声盼永恒
你在雨中我却在伽蓝觅不到你
繁华声遁入空门折煞了世人
梦偏冷辗转一生情债又牵

如你默认生死前
转山转水转佛前
等待着再见

**风格特征：** 句子“雨纷纷旧故里草木深”运用了机制2：情感的物象化表达，通过雨和旧故里的描写传达孤独。另一个例子是“伽蓝寺听雨声盼永恒”，通过伽蓝寺的意象表达对永恒的渴望，使情感表达更具层次感。

## 反例（此风格刻意回避的写法）

### 反例：晴天（周杰伦自作词）
故事的小黄花从出生那年就飘着
童年的荡秋千随记忆一直晃到现在
所以那时候才说着要去哪里哪里
怎么人连做梦都那么拼

刮风这天我试过握着你手
但偏偏雨渐渐大到我看你不见
还要多久我才能在你身边
等到放晴的那天也许我会比较好一点

从前从前有个人爱你很久
但偏偏风渐渐把距离吹得好远
好不容易又能再多爱一天
但故事的最后你好像还是说了拜拜

### 反例：以父之名（周杰伦自作词）
微凉的晨露沾湿黑礼服
教堂的钟声响起了
我宣读誓言望着你嫩白的脸庄严

圣洁的气氛
我无法呼吸
恶魔的低语
不断侵蚀我心灵

闭上眼睛祈祷离别前夕
孤独的魂魄能否接受主的洗礼

以父之名判决我
世人啊你们听好
命运无法抗拒
说好的幸福呢

## 创作指引

1. 阅读风格指纹，理解每一个维度的具体要求。
2. 研究正例中的具体句子：注意哪些词汇、哪种句法节奏、哪类意象的组合方式。
3. 用正例中出现的意象逻辑（而非正例的具体内容）来构建新作品。
4. 始终通过物象传递情感，不直白陈述「我很思念」「我很难过」。
5. 检查反例：确认你的输出中没有出现反例中被指出的那些写法。
6. 直接输出歌词/诗句，不需要解释你的创作思路。
