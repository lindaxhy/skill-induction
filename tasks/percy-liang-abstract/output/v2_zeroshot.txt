Pre-training language models data-efficiently is increasingly important as compute scales faster than available web text. While synthetic data augmentation has shown promise, its utility for modeling the original web text distribution—the core objective of pre-training—remains underexplored. We study how synthetic data can achieve lower i.i.d. validation loss on web text under an asymptotically infinite compute regime, targeting the best possible 300M parameter model given only 200M real tokens. We first demonstrate that mixing real data with synthetic rephrases monotonically improves i.i.d. loss as the number of rephrases increases, achieving an estimated $1.48\times$ data efficiency improvement over training on real data alone. To further improve loss scaling, we introduce the perspective that synthetic generations should form long megadocuments rather than independent short documents. We propose two megadocument algorithms: stitched rephrasing, which concatenates rephrases of the same document inspired by In-context Pre-training, and latent thoughts, which inserts rationales connecting document chunks inspired by chain-of-thought reasoning. Both megadocument methods outperform simple rephrasing on i.i.d. validation loss, long-context loss, and downstream benchmarks, achieving $1.64\times$ and $1.80\times$ data efficiency respectively at 32 generations, with less saturation as generation count increases. Gains are especially pronounced on long-context evaluation sets and translate to 6–9% average accuracy improvements on downstream benchmarks. Finally, all three synthetic data methods compose with ensembling, an independently data-efficient strategy, while self-distillation does not, highlighting a qualitatively distinct benefit of our proposed approaches. We open-source all experimental runs and code.
