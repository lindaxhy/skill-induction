Pre-training language models faces a growing data bottleneck as compute scales faster than available web text. We study how synthetic data augmentation can improve data-efficient pre-training by achieving lower i.i.d. validation loss on the original web distribution—the core objective of pre-training—rather than merely transforming data into higher-quality or task-specific formats. Operating in the asymptotically infinite compute regime with a fixed 300M parameter model and 200M real web tokens, we first demonstrate that mixing rephrases generated by an external language model (Llama 3.1 8B Instruct) with real data monotonically improves i.i.d. loss as the number of rephrases increases, achieving an estimated 1.48× data efficiency over training on real data alone. We then introduce the perspective that synthetic generations should form long megadocuments rather than shuffled independent documents, and propose two megadoc algorithms: stitched rephrasing, which concatenates rephrases of the same document inspired by In-context Pre-training, and latent thoughts, which interleaves rationales connecting document chunks inspired by chain-of-thought reasoning. Both megadoc methods outperform simple rephrasing, achieving 1.64× and 1.80× data efficiency respectively at 32 generations, with less saturation as generation count increases. These gains extend to long-context validation loss and downstream benchmarks, improving average accuracy by up to 9% over the real-data baseline. Finally, all three synthetic data strategies compose with ensembling, an independently data-efficient technique, while self-distillation does not, highlighting a qualitatively distinct benefit of our approach. We open-source all experimental runs and code.
