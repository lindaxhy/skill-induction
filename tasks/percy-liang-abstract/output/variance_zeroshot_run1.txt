Compute is scaling faster than available web text for language model pre-training, motivating data-efficient pre-training methods that better model the underlying web distribution. While synthetic data augmentation has shown promise, its utility for improving i.i.d. validation loss on the original data distribution remains underexplored. We study this problem under an asymptotically infinite compute regime, training 300M parameter models on 200M real web tokens with unlimited compute and an external synthetic data generator (Llama 3.1 8B Instruct). We first demonstrate that mixing real data with synthetic rephrases monotonically improves i.i.d. validation loss as the number of rephrases per document increases, achieving $1.48\times$ data efficiency over a real-data-only baseline. We then introduce the perspective that synthetic generations can form a single long megadocument rather than independent short documents, and propose two megadocument algorithms: (1) stitched rephrasing, which concatenates rephrases of the same document inspired by In-context Pre-training, and (2) latent thoughts, which inserts rationales connecting document chunks inspired by chain-of-thought reasoning. Both megadocument methods outperform simple rephrasing on i.i.d. loss, long-context loss, and downstream benchmarks, achieving $1.64\times$ and $1.80\times$ data efficiency respectively at 32 generations, with less evidence of plateauing under increasing generation count. Downstream accuracy improves by up to 9% over the real-data baseline on PIQA, SciQ, and ARC Easy. Finally, we show that all three synthetic data methods compose with ensembling, an already data-efficient strategy, while self-distillation does not, highlighting a qualitatively unique benefit of our proposed approaches. We open-source all experimental runs and code.
