Compute is scaling faster than available web text for language model pre-training, motivating data-efficient pre-training methods that better model the underlying web distribution. While synthetic data augmentation has shown promise, its utility for improving in-distribution modeling—the core objective of pre-training—remains underexplored. We study how synthetic data can achieve lower i.i.d. validation loss on web text under an asymptotically infinite compute regime, training 300M parameter models on 200M real tokens with unlimited compute using an external generator (Llama 3.1 8B Instruct). We first demonstrate that mixing real data with rephrases monotonically improves i.i.d. loss up to 32 rephrases per document, achieving 1.48× data efficiency over a real-data-only baseline. To further improve loss scaling, we introduce the perspective that synthetic generations can form long megadocuments rather than independent short documents. We propose two megadoc algorithms: (1) stitched rephrasing, which concatenates rephrases of the same document inspired by In-context Pre-training, and (2) latent thoughts, which inserts rationales connecting document chunks inspired by chain-of-thought reasoning. Both megadoc methods outperform simple rephrasing, achieving 1.64× and 1.80× data efficiency respectively, with less evidence of plateauing as generation count increases. These gains extend to long-context validation loss and downstream benchmarks, improving average accuracy by 6% and 9% over the baseline. Finally, all three synthetic data methods compose with ensembling, an independently data-efficient strategy, while self-distillation does not, highlighting a qualitatively distinct benefit of our approach. We open-source all experimental runs and code.
